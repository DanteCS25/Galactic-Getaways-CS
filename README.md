### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/DanteCS25/Galactic-Getaways-CS.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Start the server in `server` file
   ```sh
   npm start
   ```
